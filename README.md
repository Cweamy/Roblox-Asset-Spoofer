# Roblox Sound and Animation Spoofer

A powerful Python utility designed to fetch, download, and decompress Roblox assets. This tool bypasses standard limitations by utilizing the `assetdelivery` API to grab animations and audio files directly.

## Installation & Usage
1. Clone this repository.
2. Set your `.ROBLOSECURITY` cookie in your environment variables.
3. Run the script using the Python launcher:
   ```bash
   py main.py
   ```
